# Сборщик данных

Приложение, сканирующее [реестр участников бюджетного процесса, а также юридических лиц, не являющихся участниками бюджетного процесса](https://budget.gov.ru/%D0%9E%D1%82%D0%BA%D1%80%D1%8B%D1%82%D1%8B%D0%B5-%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D0%B5?code=7710568760-RUBPNUBP), и сохраняющее записи локально.

В качестве необходимых аргументов принимает даты (в формате DD.MM.YYYY):

`--lastUpdateFrom`: начало диапазона дат обновления записей на ЕПБС;

`--lastUpdateTo`: конец диапазона дат обновления записей на ЕПБС.

### Пример команды запуска:
```bash
java -jar scraper.jar --lastUpdateFrom 10.06.2026 --lastUpdateTo 15.06.2026
```
